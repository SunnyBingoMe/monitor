
BTH SNMP MONITOR v11.06

Requirements: 
1. Web Server.
2. MySQL database.

Set up:

1. Unzip this file to your web server's folder.
2. Open your browser, go to http://127.0.0.1/the unziped folder/monitor/install/.
3. Follow the steps there.
4. Have a look at user_manual.pdf first (you could find it the same folder as README.TXT).
